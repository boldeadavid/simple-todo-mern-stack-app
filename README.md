# simple-todo-mern-stack-app
This is a project for testing. Selenium + Postman. The application is created in react + node.js

Comandline to run backend in PowerShell:
$env:NODE_OPTIONS="--openssl-legacy-provider"
npm start

Comandline to run frontend in CommandPrompt:
npm start

Links: http://localhost:3000/login - for login page, http://localhost:3000/ - for tge TODO's list

For Testing: VisualStudio C# Selenium - frontend, Postman - Backend
Run tests: Test Explorer in Visual Studio, Collections->Run in Postman

Simple Login test, Add and Delete TODO's in Selenium Visual Studio C#
Simple POST /login, GET /items, POST /items, PUT /items/:id in postman (Pass & Fail coverage)
